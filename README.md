# Heart Disease Predictor

## Project Overview
This project aims to predict the presence of heart disease in patients based on various medical attributes. Utilizing machine learning algorithms, it analyzes the dataset and provides predictions for individuals.

## Features
- Predicts heart disease presence
- User-friendly interface
- Visualization of results

## Installation
To install the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/krishna200822/Heart-disease-predictor.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Heart-disease-predictor
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To use the application, run the following command:
```bash
python app.py
```

## Dataset Info
The dataset consists of various features such as age, sex, blood pressure, cholesterol levels, and other medical factors. The data is essential for training the predictive model.

## Model Details
The project uses a combination of machine learning algorithms, including:
- Logistic Regression
- Random Forest
- Support Vector Machines
The model is trained on the dataset and tuned for optimal performance.

## Project Structure
- `app.py` - main application file
- `requirements.txt` - dependencies
- `data/` - folder containing datasets
- `models/` - folder with model training scripts

## Input Parameters
The model takes in the following parameters:
- Age
- Gender
- Blood Pressure
- Cholesterol Level
- Other medical factors

This project should be used for educational purposes only. The predictions made by the model are not a substitute for professional medical advice.
